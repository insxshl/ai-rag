# AI PDF Buddy 🤖📄

AI PDF Buddy is a full-stack RAG (Retrieval-Augmented Generation) application that allows users to upload PDF documents and chat with them using AI.

Users can ask questions about uploaded PDFs, summarize content, extract key points, and get intelligent answers based on document context.

---

## Features

* User Authentication (Login/Register)
* PDF Upload & Processing
* AI-Powered Chat with PDFs
* Context-Aware Responses
* Conversation History Support
* Vector Search Retrieval
* Modern Black & Gold UI
* Responsive Design
* Secure JWT Authentication
